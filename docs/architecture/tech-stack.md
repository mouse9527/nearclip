| 数据库 | Room (Android) | 2.5.0 | 本地数据存储 | 类型安全、SQLite 抽象、迁移支持 |
| 缓存 | Core Data (Mac) | - | 本地数据存储 | Apple 生态原生、性能优化 |

| IaC 工具 | - | - | 基础设施 | 本地应用无需基础设施即代码 |

| 日志 | 平台原生日志 | - | 调试和监控 | 系统集成、性能优化、开发者友好 |
| CSS 框架 | - | - | 不适用 | 原生应用无需 CSS |
